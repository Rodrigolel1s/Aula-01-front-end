Bloco de Notas

O Bloco de Notas Online é uma ferramenta para criar anotações online. Nesta ferramenta você cria suas anotações com facilidade, sem precisar fazer login ou criar alguma conta. Para usar essa aplicação é muito fácil, digite ou cole algum texto no espaço em branco localizado no retângulo azul e ele será salvo automaticamente.
