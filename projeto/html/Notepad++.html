Notepad++

O Notepad (ou Bloco de notas em português) é um editor de texto simples que é incluído em todas as versões Microsoft Windows desde a versão 1.0 em 1985.[1] O uso mais comum do Bloco de notas é exibir ou editar arquivos de texto (.txt), mas muitos usuários o consideram uma ferramenta simples para criar e editar páginas web e scripts de linguagens de programação.
